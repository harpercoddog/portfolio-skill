# Rule-Based Suggestions

MVP 分析模块只输出规则型提醒，不做自由发挥。

## 默认规则

### 单日跌幅提醒

- 条件：`day_change_pct <= -5%`
- 输出：建议复盘最近的基本面、消息面或技术位

### 集中度提醒

- 条件：单一标的市值占比 `>= 35%`
- 输出：提示关注单一标的风险

### 浮盈浮亏提醒

- 条件：单标的未实现收益率绝对值 `>= 20%`
- 输出：提示复核止盈止损和仓位纪律

### 缺少价格提醒

- 条件：持仓存在但没有最新价格
- 输出：提示先执行 `sync_prices`

## 后续可扩展规则

- 连续多日下跌提醒
- 账户层面集中度提醒
- 行业暴露提醒
- 汇率风险提醒
